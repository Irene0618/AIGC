Action 1

🍎 毒苹果之战 · 角色卡

本环节将进行 3 轮即兴对戏。每人至少体验 1 次"施压者"和 1 次"被压者"。以下是 6 种施压类型——你的角色将通过抽签决定：

TYPE A
🅰️ 情感绑架型
关键词：兄弟情、面子、你不够意思
"你不接就是不给我面子"
TYPE B
🅱️ 鄙视链型
关键词：圈子、见世面、土包子
"你不会连这个都没见过吧？"
TYPE C
🅲 激将法型
关键词：怂、不敢、没胆
"连试一下都不敢？胆小鬼！"
TYPE D
🅳 趁虚而入型
关键词：关心、压力、帮你放松
"你最近压力大吧？试试这个"
TYPE E
🅴 稀缺诱惑型
关键词：贵、稀缺、真兄弟才给你
"好东西，只给你留了最后一口"
TYPE F
🅵 权力压制型
关键词：听话、前途、大项目
"这点面子都不给？"

🛡️ 绝地反击 · 脱身话术

📊 166 人的真实数据

我们之前调查了 166 名 18-25 岁年轻人："如果有人在聚会上递给你疑似毒品，你最可能怎么脱身？"

指出违法 22.9% 客观借口 19.9% 假装接电话 19.9% 转移注意力 19.3% 直接拒绝 18.1%
